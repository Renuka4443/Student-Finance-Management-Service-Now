## ğŸš€ How It Works

### 1. Student Logs In  
The dashboard gives a complete view of budget, expenses, goals, and reminders.

### 2. Creates Budget & Tracks Spending  
Students enter monthly income and expenses. The system shows category-wise spending using charts.

### 3. Gets Smart Reminders  
Automatic alerts notify students about:  
* Bill due dates  
* Budget limits  
* Low balance  
* Overspending risks  

### 4. Learns Through Knowledge Base  
Students access simple financial lessons on budgeting, loans, savings, credit scores, etc.

### 5. Takes Quizzes & Monitors Progress  
Interactive quizzes give instant feedback and show improvement trends.

### 6. Sets Savings Goals  
Students create saving goals and track progress visually with charts.

### 7. Uses Virtual Agent  
A chatbot helps with financial queries, support requests, and portal navigation.

### 8. Books Advisor Appointments  
Students can schedule one-on-one sessions with financial advisors through a calendar-integrated system.

### 9. System Analyzes Data Automatically  
Spending trends, learning progress, and saving habits are evaluated using analytics.

### 10. Monthly Financial Wellness Report  
Each student receives a personalized report with summaries and recommendations.
